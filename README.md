# ⚡️ Hermes Quant System — 爱马仕量化交易系统

**自动化合约交易 | 飞书实时推送 | 24/7 无人值守**

## 🚀 核心能力

### 1. 自动化合约交易
- **1分钟 K 线** 实时扫描，RSI + EMA 双指标信号
- **趋势信号**（EMA向上+RSI低位做多 / EMA向下+RSI高位做空）
- **极端信号**（RSI超买超卖直接入场）
- ATR 波动率过滤，避免僵尸币
- 动态合约张数计算（`GetContractValue()`）
- 移动止盈止损 + 保本激活

### 2. 飞书实时推送
- 每5分钟推送持仓报告
- 开仓/平仓即时通知
- 盈亏变动预警
- 异常状态告警

### 3. 24/7 看门狗
- 每2分钟检测机器人状态
- 异常时自动重启
- 独立于本地机器运行
- 断网/息屏不受影响

### 4. AI API 聚合（OpenRelay）
- Claude Desktop / VS Code Copilot 配额复用
- 多供应商故障转移
- 统一 API 端点
